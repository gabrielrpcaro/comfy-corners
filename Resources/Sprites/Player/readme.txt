Sistema de Naming
Para garantir que os sprites sejam facilmente identificáveis e organizados, usaremos um sistema de naming consistente. Aqui está um exemplo de como nomear os arquivos de sprite:

plaintext
Copiar código
{part}_{type}_{style}_{color}.png
Por exemplo:

Para um cabelo masculino, estilo 1, cor 1: Hair_Masculine_Style1_Color1.png
Para um torso feminino, estilo 2, cor bronze: Torso_Feminine_Style2_Bronze.png
Para olhos estilo 3, cor azul: Eyes_Style3_Blue.png
